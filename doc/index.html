ThreeContext

ThreeContext creates a WebGL context using THREEjs. It also handle mouse control. An event can be associated to a ThreeContext instance: onRaycast with the method .on("onRaycast", function(s){...}) where s is the section object being raycasted.

new ThreeContext(divObj: DONObject)

Extends EventManager

Parameters
divObj (DONObject = null) the div object as a DOM element. Will be used to host the WebGL context created by THREE
